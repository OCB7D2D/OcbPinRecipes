Description
===========
レシピをゲーム画面上にピン留めします。
アイテムを作成するときに、どの素材がいくつ必要なのか一目で分かります。
7 Days to Die Alpha20.x対応です。

GitHubにocbMaurice氏のOcbPinRecipesというオープンソースのプログラムがあったので、
コンパイルして日本語専用に作り直しました。
 https://github.com/OCB7D2D/OcbPinRecipes

 1. インベントリ画面を開いてアイテムを選択すると、「ピン留め」ができます。
 2. ピン留めされたレシピを操作したり、ピン留めを解除する場合は、インベントリ画面を開いてください。
 3. 素材が必要数揃うと表示されるハンマーアイコンをクリックすると、そのアイテムを作成できます。
 4. ＋アイコンをクリックすると作成したい数を増やせます。
 5. 作成したい数が2つ以上設定されている場合は、表示される－アイコンで作成数を減らせます。


もちろん、拙作のNoob-HUDとの相性もバッチリです。まだの人は使ってみてくださいね。

Install
=======
\Steam\steamapps\common\7 Days To Die\フォルダにzip内のModsフォルダをそのまま解凍＆コピーまたは上書きしてください。


Uninstall
=========
アンインストールしたい場合は、
\Steam\steamapps\common\7 Days To Die\Mods\フォルダ内のReciPinフォルダを削除してください。
